ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ:
Запись файлов. Класс FileWriter
Класс FileWriter является производным от класса Writer. Он используется для записи текстовых файлов.
Чтобы создать объект FileWriter, можно использовать один из следующих конструкторов:

    FileWriter(File file)
    FileWriter(File file, boolean append)
    FileWriter(FileDescriptor fd)
    FileWriter(String fileName)
    FileWriter(String fileName, boolean append)

Так, в конструктор передается либо путь к файлу в виде строки, либо объект File,
который ссылается на конкретный текстовый файл. Параметр append указывает,
должны ли данные дозаписываться в конец файла (если параметр равен true), либо файл должен перезаписываться.
